# FFX Keyword Demand Map

**Source of truth for keyword-driven generation.** Generation targets a keyword from
this map, then pulls a video/KB passage that *serves* that target — never one-video-one-
article. This replaces the GSC-echo target source (`intelligence:brief` fed by the site's
own zero-volume queries), which reinforced the exact terms nobody searches.

## The end goal (drives every choice)

FFX has zero traffic. The chain is: **get indexed → get found on a real query → teach
Salman's method → convert to Discord / bootcamp / VIP.** Articles are the *acquisition
channel*, not the product. A target only belongs here if a real person searches it, a
low-authority expert site can win it, and answering it well moves the reader toward the
method and the offer.

## The linchpin: private vocabulary → search vocabulary

Salman already has the knowledge; it is labelled with words nobody searches. **Rank for the
searched term; teach the proprietary name inside the article.**

Volume bands (public-signal estimates — autocomplete/PAA/Reddit/YouTube; exact MSV needs a

Winnability is for a low-DR site run by a genuine expert. **AIO** = survives AI Overviews
(drives clicks despite an AI answer).

## TIER 3 — AIO-resilient experience / moat plays (highest differentiation)

These are the ones AI Overviews and content farms **cannot** fabricate — real trades, real
losses, original data, branded methodology. Highest conversion, hardest to copy.

- "how I passed a $100k prop firm challenge" — lived experience + proof
- "[method] review after 6 months" — temporal + first-hand result
- "does supply and demand actually work in forex" — skeptic/validation intent
- "biggest mistakes I made in my first year trading forex" — relatable experience
- "my [N]-trade breakdown of [setup]" — original data (only if genuinely logged)
- "catch the wick / momentum candle setup" (branded) — owns the term, forces citation

## EXCLUDE — do not target

- **Head terms** (unwinnable at low DR): forex trading, how to trade forex, best forex
  broker, forex signals, forex strategy, what is forex.
- **Definitional terms** (dead clicks — AIO answers them in-paragraph at ~91% coverage,
  ~60% CTR collapse): what is a pip, what is leverage, what is a lot size, what is spread,
  what is a stop loss. Use only as internal-link support pages, if at all.
- **The site's own zero-volume vocabulary as a primary target**: "opening candle
  continuation strategy", "momentum candle", "time fractal". Keep them as brand anchors and
  internal links; never chase traffic with them.

## How generation reads this map

1. Select an **open** Tier-1 target (highest priority first).
2. Pull the video/KB passage(s) that genuinely **serve** that target — many sources may
   enrich one canonical page. Never mint one article per video.
3. Write to the **searched term** (title, one H2, first 100 words, naturally in body); teach
   the **proprietary name** alongside it ("the setup I call the momentum candle — the
   textbook term is displacement").
4. Demonstrate first-hand experience **only where the source genuinely supports it**; never
   fabricate a trade, win-rate, or statistic.
5. Gate before it can ever publish (anti-fabrication hard-fail, trust ≥70, similarity ≤0.90,
   structural ≤0.55, voice via the existing preventive system).
6. On publish, mark the target **claimed** so the corpus diversifies instead of clustering.
